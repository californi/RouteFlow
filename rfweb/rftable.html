The table shows the current associations of VMs and datapaths in the RouteFlow network.
  • A row with all columns filled means an active entry.
  • A row with only a VM ID represents a registered, idle and never used VM.
  • A row with only a DP ID represents a registered, idle and never associated datapath.
  • A row with only a VM ID and DP ID represents an association that is inactive.
    The reason could be an offline datapath.
